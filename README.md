# ToDoList-MariusLEORAT-CDOF6

## Description
ToDoList is a console-based application designed to help users manage their tasks efficiently. With this lightweight tool, you can add, delete, and mark tasks as completed, making it ideal for managing personal or small project task lists.

## How to Run the Project

### Prerequisites
- Ensure you have Python 3.x installed on your machine.

### Steps
1. **Clone the repository to your local machine:**
   ```bash
   git clone https://github.com/mariusokoto/ToDoList-MariusLEORAT-CDOF6.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd ToDoList-MariusLEORAT-CDOF6
   ```
3. **Run the application:**
   ```bash
   python toDoList.py
   ```

## Features
- **Add Tasks**: Easily add new tasks with a description.
- **Delete Tasks**: Remove tasks by their ID.
- **Mark Tasks as Completed**: Toggle the completion status of tasks.
- **View Tasks**: Display a list of all tasks with their status.
